About

VideoMancer v1.0.0

Detect and download videos from any web page. Supports HLS, DASH, MP4, WebM, and more.

Supported formats: MP4, WebM, MKV, AVI, MOV, FLV, M4V, TS, OGV, 3GP
Streaming protocols: HLS (m3u8), DASH (MPD)
Audio formats: MP3, AAC, OGG, WAV, FLAC, M4A, OPUS
